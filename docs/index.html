Eldritch Horror: Random Field Generator

Get random fields!

What is this for?

This is a small tool to determine a random field in the board game Eldritch Horror. While playing this game it may occur that a random field must be chosen, in which case it can get complicated since

some are named and others are numbered. This tool provides a simple and neat solution to this matter.
